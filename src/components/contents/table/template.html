参数

参数 说明 类型 可选值 默认值
extraTools 额外的头部工具按钮,格式遵循tools的格式,框架会为你额外添加到tools里面 — — null
tools 默认有添加,修改,删除3个按钮, 如果您只需要添加,或者3个中的2个,请通过toolsFlag设置 Array — —
dialog 设置添加,修改的时候弹出对话框的属性 Object — null
rowTools 行按钮 Array — []
toolsFlag 采用二进制移位算法,1=添加,2=修改,4=删除 Number — —
enableSearch 是否允许搜索,这个选项关系到是否会显示出搜索栏 Boolean true/false true
searchPlaceholder 这个选项可以设置搜索框内提示文字,不同的表格搜索条件是不同的 String — 输入搜索
sourceUrl 远程地址,获取数据用的,这个选项优先于source选项 Array Array/null null
source 本地json数据 Array Array/null null
searchUrl 点击搜索按钮的时候，走这个地址去检索， 这个值主要是为了解决如下的场景： 第一次加载的时候是source本地数据，然后点击搜索的时候去搜索远程数据 需求很变态，举个例子，流程中甲方是a和b，要显示个列表，第一次把a，b显示出来，点击搜索的时候去后端调取所有甲方的数据 注意： 这个值不要轻易去设置，除非你遇到了上述场景，因为它会去覆盖掉sourceUrl的值，并且会导致source设置无效 String String/null null
columns Array Array/null null
pagination 是否显示分页 Boolean true/false true
memorable 是否带有搜索记忆功能,存储前几次搜索选中的数据, 记忆的数据不参与分页 Boolean true/false false
noCache 不需要缓存, 针对memorable属性的记忆功能 如果表格设置了memorable,但是有时候确实是不需要记忆功能 比如dialog操作结束之后,需要自动reload,此时的reload不需要将选中的数据记忆下来 Boolean true/false false
remoteParam 自定义远程传递参数 Object 一 null
pager 分页参数设置，pageSize : 每页显示条数，curPage: 当前页数，total: 总条数, totalPage: 总页数， 值为Number类型 Object 一 一
singleSelected 是否单选,默认是多选 Boolean true/false false
searchFields 这个选项是存储搜索关键字的, 一般不需要设置这个参数,而是框架用来存储用的, 当然,如果设置了,那么框架会把你的设置作为额外的设置,一起进行判断 Array 一 一
isEmptyOnInit 有时候列表需要在初始化的时候不加载数据,填入搜索条件的时候再加载,这个选项可以帮助你 Boolean true/false false

事件

事件 说明 参数
clickRowEvent 点击某一行事件, 特别注意: 1. 单选的时候,取消选中不会触发这个事件 2. 行按钮点击或者是某个单元格的点击事件不会触发这个事件 clickRowEvent: function(event, index, row, isSelect) {}， event: 点击事件参数， index: 这一行的下标从0开始， row: 本行数据对象， isSelect: 是否选中， true : 选中，false : 取消选中
selectRowEvent 选中某一行事件, 这个和上面的区别就是只有选中才会触发这个事件， 特别注意: 1. 行按钮点击或者是某个单元格的点击事件不会触发这个事件 selectRowEvent: function(event, index, row) {}， event: 点击事件参数， index: 这一行的下标从0开始， row: 本行数据对象，
